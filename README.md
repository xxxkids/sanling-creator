# 三领漫剧

三领文化 AI 影视生产工具。基于 moyin-creator (AGPL-3.0) 二次开发。

## 工作流

```
剧本导入 → 角色校准 → 场景校准 → 导演分镜 → 视频生成 → 语音配音 → 导出
```

## 板块

| 板块 | 功能 |
|------|------|
| 概览 | 项目元数据总览 |
| 剧本 | 导入/粘贴剧本，AI 解析角色+场景+分集 |
| 角色 | 角色库（6层身份锚点+多视图+服装变体） |
| 场景 | 场景库（建筑/光影/色彩/道具） |
| 导演 | 分镜编辑（支持三领导演 v6.1 分镜表导入） |
| 视频 | Agent 视频生成（分镜草图+分批+回滚） |
| 语音 | 角色 TTS 配音（火山引擎） |
| 资产 | 资产管理 |
| 素材 | 媒体库 |
| 导出 | 成片导出 |

## 技术栈

- Electron + React + TypeScript
- Zustand 状态管理
- Tailwind CSS
- 火山引擎 API（DeepSeek / Seedream / Seedance / TTS）

## 开发

```bash
npm install
npm run dev
```

## 与 Hermes Agent 协作

```
三领编剧大师 → 剧本创作
     ↓
三领导演 v6.1 → 分镜表
     ↓
三领漫剧 APP → 资产入库 + 视频生成 + 配音
```
